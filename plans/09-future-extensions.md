# Feature Plan: Future Extensions (DONE)

## Objective
Define clear post-v1 evolution paths without blocking MVP delivery.

## Candidate Features
- Team/multi-user aggregation
- Budget alerts and threshold notifications
- Scheduled exports and reporting
- Provider-level comparisons
- Multi-machine sync/import
- Plugin hooks for custom metrics

## Deferred Decisions
- Authn/authz model for hosted web
- Persistence for precomputed analytics
- Event-stream architecture vs periodic polling

## Exit Criteria for Considering Extensions
- CLI stable
- API parity achieved
- Web dashboard validated by real usage
- Baseline performance metrics established
